# Site estático: 🎥 IMDB  
  
## Objetivo: 
O desafio é desenvolver a página inicial de um site de avaliação de filmes (IMDB - Internet Movies Database) utilizando apenas **HTML5** e **CSS3**.

## Tópicos:
Neste desafio, você vai praticar os seus conhecimentos em:
